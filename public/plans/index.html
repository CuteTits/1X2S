What's your API response time in production?

+

P50 latency under 50ms globally. Production plan guarantees P99 under 50ms. Enterprise deployments can achieve sub-10ms with dedicated infrastructure.

Can we deploy on-premise or in our own VPC?

+

Yes. Production plan supports AWS VPC deployment. Enterprise can run fully on-premise with your own infrastructure, or in any cloud environment.

What authentication methods do you support?

+

API keys, OAuth 2.0, SAML 2.0 (Enterprise), mTLS certificates, and JWT. All support rotate without downtime. Enterprise includes SSO integration.

+

Developer plan has fixed limits. Production and Enterprise support custom rate limits and burst capacity. We'll work with you to scale before hitting limits.

How is billing calculated? Can we commit to annual usage?

+

Fixed tier pricing for Developer/Production. Enterprise has usage-based or committed pricing. Annual commitments provide 20-30% savings. No overage charges.

Do you support data residency compliance?

+

Production plan supports regional data residency. Enterprise can enforce data locality, GDPR compliance, HIPAA requirements, and custom compliance frameworks.
