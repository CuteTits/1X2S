Static 2.0

Static 2 is a Fine-Tuned AI Model is built on Mistral 7B and optimized with QLoRa for advanced sports analytics. It integrates real-time/live-search capabilities with 2 billion additional football-specific knowledge points, covering historical data and live video feed analysis. Designed to run efficiently on AWS servers with an environmentally conscious footprint, it is accessible via both chatbot interface and API, delivering dynamic sports intelligence for analysts, coaches, and enthusiasts.
