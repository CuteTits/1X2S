Privacy & Architecture

Enterprise-grade privacy, scalability, and modular design

lock Privacy Model

  • No Query Storage: User queries are not logged or retained
  • No Input Logging: Requests are processed without persistent storage
  • No API Key Logging: Credentials are kept confidential
  • Anonymous Metrics: Only aggregated, anonymous monitoring for system health
  • Third-Party Compliance: Third-party APIs follow their own privacy policies

Privacy-first architecture is core to 1EX's positioning and design.

architecture Technical Architecture

  • API-First Design: REST/GraphQL endpoints for seamless integration
  • Async-First: I/O optimized for high concurrency and low latency
  • Python-Based: FastAPI recommended for production deployments
  • Modular Sources: Pluggable retrieval adapters for flexibility
  • Horizontally Scalable: Stateless design for cloud deployment
  • Redis Caching: High-performance result caching layer
  • Optional PostgreSQL: Metadata persistence and analytics
